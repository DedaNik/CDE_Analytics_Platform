CDE Analytics Dashboard (СОД)
Описание

BI-отчёт в Power BI для анализа процессов согласования документации и обработки замечаний в системе СОД (Common Data Environment) в строительных проектах.

Задача

Отсутствие централизованной аналитики:

данные распределены по объектам
анализ выполнялся вручную
невозможно оценить процессы на уровне портфеля

Решение: создание единой аналитической модели и дашборда для контроля процессов.

Данные
23 000+ согласований
50 000+ замечаний
100+ объектов
источники: CSV (files, issues, reviews, users), MySQL
Стек

Power BI, DAX, Power Query, Data Modeling, ETL

Реализация
объединение нескольких источников данных
построение модели (files / approvals / issues / users)
разработка KPI (время согласования, SLA, загрузка)
интерактивный дашборд с фильтрацией и drill-through
Результат
−30% времени на анализ данных
−25% ошибок

70 000 записей в едином интерфейсе

рост пользователей: +100/месяц
Аналитика
статус и динамика согласований
обработка замечаний
качество документации
загрузка сотрудников
временные показатели процессов
Вывод

Реализована система BI-аналитики, заменившая ручной анализ и обеспечившая прозрачность процессов согласования и контроля качества документации.
